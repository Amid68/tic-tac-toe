# Tic-Tac-Toe Game

This is a simple C-based Tic-Tac-Toe game where you play against the computer.

## How to Play
1. The player is 'X', and the computer is 'O'.
2. Enter the row and column numbers (1-3) to make your move.
3. The game alternates between the player and the computer until there is a winner or a tie.
4. After the game ends, you can choose to play again or exit.

## Compilation and Execution
To compile and run the program:

```bash
gcc -o tictactoe main.c
./tictactoe
```

Enjoy the game!
